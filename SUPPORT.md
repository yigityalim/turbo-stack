# Support

This is a private project maintained by the core team.

- **Bugs / feature requests:** open an issue using the templates in
  `.github/ISSUE_TEMPLATE/`.
- **Security issues:** do not open a public issue — see SECURITY.md.
- **Direct contact:** <contact@mehmetyigityalim.com>

Before opening an issue, please:

1. Confirm you are on the latest `main`.
2. Run `bun install` and reproduce with a clean build (`bun run build`).
3. Include environment details (OS, Bun version) and reproduction steps.
